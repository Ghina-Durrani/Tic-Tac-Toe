# 🎮 Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript. This project allows two players to play turn-by-turn on a 3×3 grid, with automatic win and draw detection.

## 🚀 Features
- Two-player gameplay (X and O)
- Win and draw detection
- Simple and responsive UI
- Restart game option

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🎮 How to Play
Click on the boxes to place X or O. The first player to align three marks in a row, column, or diagonal wins.
